# 追格小站点评主题（开源版）

### 主题概述
追格小站点评主题是一个互联网创新产品分享与点评导航主题（基于WordPress开发）。用户在这发掘和发布有趣的产品（网址、小程序、APP），为喜欢的点赞与点评。响应式布局，专业版自带前端会员中心（用户认证、发布、个人主页、消息等），并支持微信扫码登录。
